# OOP Expense Tracker 💰

A command-line Expense Tracker built with Python using 
Object Oriented Programming (OOP) principles.

## Features
- Add expenses with category, description and amount
- View all expenses
- Calculate total amount spent
- Delete expenses by description

## How to Run
1. Make sure Python is installed
2. Clone this repository
3. Run the command: python oop_expense_tracker.py

## Tech Used
- Python 3
- OOP Concepts (Classes, Objects, Encapsulation)

## What I Learned
- Designing multiple classes that work together
- Using _init_ for object initialization
- Implementing Encapsulation to structure data cleanly
- Building real world logic using OOP principles
